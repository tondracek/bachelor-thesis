\chapter{Hardwarové a softwarové požadavky aplikace}\label{ch:requirements}
Tato příloha shrnuje minimální požadavky na zařízení pro běh mobilní aplikace \textit{My Farmer}.


\section{Softwarové požadavky}

\begin{itemize}
    \item Minimální verze systému: Android 8.1 (API 27)
    \item Cílová verze systému: Android 16 (API 36)
    \item Požadována přítomnost služeb Google Play (Google Play Services)
\end{itemize}


\section{Hardwarové požadavky}

\begin{itemize}
    \item Podpora OpenGL ES 3.0 nebo vyšší
    \item Připojení k~internetu (Wi-Fi nebo mobilní data), které je nezbytné pro většinu funkcí aplikace
\end{itemize}


\section{Podporované architektury}
Aplikace podporuje následující procesorové architektury:

\begin{itemize}
    \item arm64-v8a
    \item armeabi-v7a
    \item x86
    \item x86\_64
\end{itemize}


\section{Volitelné funkce zařízení}
Některé funkce aplikace mohou využívat volitelné hardwarové prvky zařízení:

\begin{itemize}
    \item Lokalizační služby (GPS nebo síťová lokalizace)
\end{itemize}

Tyto funkce nejsou nezbytné pro základní běh aplikace.


\section{Omezení kompatibility}
Aplikace není určena pro zařízení:

\begin{itemize}
    \item bez podpory OpenGL ES 3.0
    \item bez služeb Google Play
\end{itemize}
