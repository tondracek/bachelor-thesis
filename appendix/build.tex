\chapter{Sestavení a spuštění aplikace}\label{ch:build}
Pro sestavení a spuštění aplikace ze zdrojových kódů je nutné mít k~dispozici následující:

\begin{itemize}
    \item Vývojové prostředí Android Studio (verze 2025.3.4 nebo novější)
    \item Připojení k~internetu pro stažení projektových závislostí a potřebných komponent SDK
\end{itemize}

Po stažení souboru \texttt{code.zip} z~archivu bakalářské práce a jeho rozbalení otevřete projekt v~prostředí Android Studio.
Následně dojde ke stažení potřebných závislostí a konfiguraci projektu.

Aplikaci bude následně možné spustit:

\begin{itemize}
    \item na fyzickém zařízení s~Androidem 8.1 (API 27) nebo novějším, s~povoleným režimem pro vývojáře a laděním přes USB
    \item nebo pomocí emulátoru v~prostředí Android Studio
\end{itemize}

Pro správnou funkčnost aplikace je vyžadováno:

\begin{itemize}
    \item připojení k~internetu
    \item splnění minimálních systémových požadavků cílového zařízení uvedených v~příloze~\ref{ch:requirements}
\end{itemize}
